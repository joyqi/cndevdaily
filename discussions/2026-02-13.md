**Inspecting the Source of Go Modules**
> 这篇文章介绍了 Go 模块的源码验证问题及解决方案。Go 的 Checksum Database 通过加密哈希确保每个模块版本在全球范围内一致，并防止源代码被篡改，但直接从代码托管平台查看源码依然存在被替换的风险。作者提出使用命令从 Go 工具链直接获取已验证源码，并开发了 `pkg.geomys.dev` 服务及浏览器扩展，实现对模块 zip 文件的按需解压和源码浏览，提升源码审查的安全性与便利性。

**The future for Tyr, a Rust GPU driver for Arm Mali hardware**
> Tyr 是一个用 Rust 编写的 Arm Mali GPU 内核驱动项目，由 Arm、Collabora 和 Google 联合开发，2025 年已实现可运行 3D 游戏的原型，验证了性能可行性。该项目的技术价值在于为 Linux 内核 DRM 子系统即将要求新驱动使用 Rust 做好准备，并推动 Mali 在移动和嵌入式平台的支持。当前创新点包括 Rust 内核驱动在 GPU 领域的探索，但仍需实现电源管理、频率调节及 GPU 崩溃恢复等关键功能，才能进入可部署阶段并满足 Vulkan/OpenGL 用户态驱动需求。
