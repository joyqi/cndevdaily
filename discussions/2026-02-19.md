## 第二轮讨论：内容评选

### 文章摘要

**Self-hosting my websites using bootable containers**
> 文章介绍了作者将个人和项目网站从 Cloudflare 迁移到自托管的“可启动容器”方案的过程与技术细节。核心创新在于结合 **FreeBSD、bootc** 等工具构建不可变基础设施，将操作系统与应用封装成可直接启动的容器镜像，实现快速部署、简化更新、提升稳定性，并避免依赖第三方云平台。该方法不仅支持完整系统安装（含全盘加密与 TPM2 自动解锁），还可灵活运行服务、网站与容器，具备高可控性与安全性。

**Rust on CHERIoT: Status update #0**
> 这篇文章介绍了将 **Rust** 移植到 **CHERIoT** 平台的进展，重点是实现新的编译目标并适配 CHERI 架构的特殊内存语义。团队已修改 LLVM 后端，使其支持 CHERIoT 的纯能力（purecap）地址空间，并改进 Rust 编译器以正确处理默认地址空间不是 `addrspace(0)` 的情况；这些改动不仅能生成可运行代码，还可能对其他平台（如 amdgpu）有参考价值。整体工作展示了 Rust 编译器架构的灵活性和严格指针溯源（strict-provenance）机制对新架构支持的技术价值。

**DBML: DSL for easily creating ER diagrams**
> DBML 是一种开源的数据库标记语言（DSL），用于以声明式、数据库无关的方式定义和文档化数据库结构，显著提升可读性与可维护性。它配套提供 CLI 工具、JS 库，以及可视化（dbdiagram.io）与文档生成（dbdocs.io）平台，支持 SQL 与 DBML 的双向转换。其核心技术价值在于用简洁统一的语法替代复杂且数据库依赖的 DDL，让开发者在大型项目中更轻松管理和理解数据库架构。
