## 第二轮讨论：内容评选

### 文章摘要

**Supply Chain Attack on Axios**
> 文章揭示了一次针对 JavaScript 生态中高流行度 HTTP 客户端 Axios 的供应链攻击，攻击者在 npm 发布的 axios@1.14.1 和 axios@0.30.4 中引入了恶意依赖 plain-crypto-js@4.2.1，该依赖包含多阶段恶意载荷（包括远程访问木马）。此次发布未按正常 GitHub 标签流程进行，疑似通过失控的 npm 发布权限和长期存在的 npm token 实现。创新点在于攻击精准投毒两个版本分支并通过极小改动实现隐蔽传播，强调了对发布流程安全、依赖监控与权限管理的紧迫性。

**cocoa-way: Native macOS Wayland Compositor written in Rust using Smithay**
> Cocoa-Way 是一个用 Rust 和 Smithay 编写的原生 macOS Wayland 合成器，通过 Unix 套接字直连 Linux 应用，无需虚拟机，支持 HiDPI 和硬件加速 OpenGL 渲染，具备原生窗口装饰与优化 UI。它通过与 waypipe-darwin 配合，在 macOS 上高效显示来自 Linux 环境的图形应用，显著降低延迟，原生集成度高。未来将扩展多平台后端（Windows、Android）、多显示器支持及剪贴板同步。

**Trivially breaking confidential VMs**
> 文章介绍了 **BadAML** 攻击，它利用宿主机提供的 ACPI 表（包含 AML 字节码）在机密虚拟机内部获得任意代码执行权限，从而绕过硬件强制的内存隔离。研究团队在基于 CVM 的 Kubernetes 运行时 **Contrast** 上完整复现了该攻击，并通过构建 **AML 沙箱**，限制 AML 仅能访问共享内存页，有效封堵漏洞。该工作揭示了 ACPI/AML 在机密计算环境中的高风险访问能力，并提供了可在多数 CVM 产品中借鉴的安全缓解方案。
