### 文章摘要

**Bitwarden CLI Compromised in Ongoing Checkmarx Supply Chain**
> Bitwarden CLI 的 npm 包 (@bitwarden/cli 2026.4.0) 被发现遭受与 Checkmarx 供应链攻击相关的入侵，攻击者利用被劫持的 GitHub Actions 工作流注入恶意代码（bw1.js）。该载荷可窃取 GitHub、云服务与 SSH 凭据，并通过 npm 与 GitHub 传播，展现出高度自动化的供应链渗透能力。此事件揭示了 CI/CD 安全在开源生态中的系统性漏洞，凸显对依赖安全扫描与持续监控技术的迫切需求。

**Borrow-checking without type-checking**
> 这篇文章介绍了一种在动态类型语言中实现“借用检查”的新方法，无需静态类型系统即可确保内存与可变性安全。作者通过在栈上存储引用计数、仅在动态类型函数帧中触发检查，达到了低开销、可提供明确错误信息的运行时借用验证机制。这种设计探索了在动态语言中引入 Rust 式所有权与借用语义的可能性，为动态与静态类型系统的融合提供了一种创新思路。

**Show HN: Tolaria – open-source macOS app to manage Markdown knowledge bases**
> Tolaria 是一款开源的 macOS 应用，用于管理基于 Markdown 的知识库，强调完全本地化、文件可移植性与 Git 版本控制。其设计原则包括 *files-first*、*offline-first*、*AI-first but not AI-only* 等，确保数据所有权与可扩展性。技术上采用 Tauri + React + TypeScript 构建，支持 AI 助手集成和键盘优先的高效操作，体现了为高阶用户打造的开放、无锁定知识管理创新思路。
