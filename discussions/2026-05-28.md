### 文章摘要

**Raft Consensus with a Minority of Nodes**
> 这篇文章提出对 Raft 共识协议的非传统修改，使系统可在少于半数节点活跃的情况下继续取得进展，前提是活跃节点满足特定组合约束。作者借鉴了“Spot It!”纸牌游戏背后的有限射影平面数学结构，用这种几何构造替代传统的多数重叠原则，从而在理论上实现更灵活的共识冗余模式。这一创新展示了分布式一致性与组合数学之间的独特交叉应用。

**readable.css**
> **总结：**  
readable.css 是一个专注于提升网页可读性的轻量级 CSS 框架，它提供统一、美观的默认样式（如响应式布局、暗色/浅色模式、排版节奏与原生字体支持），而不引入动画、工具类或自定义字体等复杂特性。其核心创新在于通过语义化 HTML 自动推断样式，实现一致且简洁的视觉体验，适合重视内容呈现与写作自由的开发与创作场景。

**I think Anthropic and OpenAI have found product-market fit**
> 文章指出，Anthropic 与 OpenAI 近期通过将企业用户的订阅模式改为与 API 用量挂钩的定价，实现了真正的产品市场契合（product-market fit）。这一策略意味着两家公司成功将高频使用的编码与通用智能体产品（如 Claude Code、Codex）转化为高利润的企业收入来源，标志着从大众订阅向可持续企业模型的转型。技术价值在于基于真实使用量的定价模型推动了 LLM 商业化的成熟阶段，也为即将到来的 IPO 奠定了盈利基础。
