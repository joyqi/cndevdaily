### 文章摘要

**Full Disclosure: 1-Click GitHub Token Stealing via a VSCode Bug**
> 通过挖掘 VSCode Webview 的跨域隔离机制，作者发现了一个仅需一次点击即可窃取用户 GitHub OAuth Token 的漏洞。利用该 token 能读写所有私有仓库，而攻击者只需诱导访问者点击恶意链接，利用 postMessage 通信缺陷绕过沙箱保护，实现一键令牌窃取。这项研究突显了现代编辑器在浏览器端集成时的安全边界弱点，并强调了对第三方组件审计的紧迫性。

**Codex Discovered a Hidden HTTP/2 Bomb**
> Codex 发现了一种新型 HTTP/2 拒绝服务攻击，它巧妙地将压缩炸弹与 Slowloris 流控保持技术结合，利用 HPACK 表项的管理开销（而非解码后大小）实现内存放大，从而绕过基于头部大小或字段数的现有防护。针对 Apache httpd 和 Envoy 的创新利用打破了 RFC 允许的 Cookie 分割特性，以极低带宽将服务器内存瞬间撑至数十GB。该工作不仅揭示了对 88 万以上主流服务器的实际威胁，也展示了自动化漏洞发现工具在组合已知攻击模式方面的独特价值。

**Microsoft CEO: We’re moving from OS and apps to agents instead**
> 微软与高通合作推出 Project Solara 芯片到云平台，标志着计算范式的根本转变：从围绕操作系统和应用构建转向以 AI 代理为先。该平台将硅、软件与云端深度融合，实现更加个性化、情境感知且始终相伴的智能体验，为未来的代理优先计算奠定技术基础。
