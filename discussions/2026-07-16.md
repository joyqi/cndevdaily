> 
> **#24 [Full disclosure: Arbitrary code execution in Cursor]**

> 
> **#3 [FreeBSD 16 Retires The Last Of Its GPL Code From Its Base System]**  
> 彻底去 GNU 化？内核里最后那点 GPL 代码（估计是 dtrace 或者早年 toolchain 残留）被干掉了。对于纯粹 BSD 信徒来说，这是意识形态的胜利。从技术角度，这背后往往意味着重写或引入 MIT 替代品，比如可能用 elftoolchain 彻底替代 binutils，每个替换都值得细看实现细节。

**Command Line Interface Guidelines**
> 这篇开源指南的核心在于将传统 UNIX 哲学与现代交互设计结合，推动命令行从“机器优先”转向“人性化”的文本界面。其技术创新点在于提炼出一套适应现代开发习惯的 CLI 设计原则，如多级命令、工作流自动化等模式，旨在提升工具的易用性与愉悦感。
